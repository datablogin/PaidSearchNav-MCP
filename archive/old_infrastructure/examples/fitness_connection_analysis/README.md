# Fitness Connection Analysis Example

This directory contains example bulk action CSV files and analysis reports for Fitness Connection, a retail fitness business with multiple locations.

## Files

- `bulk_actions_campaign_optimization.csv` - Campaign optimization actions including budget adjustments and Performance Max campaign creation
- `bulk_actions_ad_group_optimization.csv` - Ad group creation and optimization with location-specific targeting
- `bulk_actions_keyword_optimization.csv` - Keyword additions with proper match types and location targeting
- `bulk_actions_negative_keywords.csv` - Negative keyword additions to prevent wasted spend
- `bulk_actions_demographic_bid_adjustments.csv` - Age and gender bid adjustments based on performance data
- `fitness_connection_optimization_report.md` - Comprehensive analysis report with recommendations

## Purpose

These files demonstrate:
1. How PaidSearchNav can generate actionable Google Ads bulk import files
2. Location-based campaign structuring for retail businesses
3. Performance-driven optimization recommendations
4. Proper negative keyword implementation to reduce wasted spend

## Usage

These CSV files can be imported directly into Google Ads using the bulk import feature. Each file targets specific optimization areas and follows Google Ads bulk import formatting requirements.
