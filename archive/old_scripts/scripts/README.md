# Scripts Directory

This directory contains analysis scripts and tools for various PaidSearchNav use cases.

## Cotton Patch Analysis (`cotton_patch_analysis/`)

Scripts for analyzing Cotton Patch Cafe data and generating comprehensive reports:

- `create_comprehensive_analysis.py` - Creates comprehensive analysis from S3 data
- `execute_production_pipeline.py` - Production pipeline execution
- `run_cotton_patch_analysis.py` - Main Cotton Patch analysis runner
- `run_individual_analyzers.py` - Individual analyzer execution
- `test_cotton_patch_parsers_analyzers.py` - Parser and analyzer testing
- `test_full_workflow.py` - Full workflow testing

## Google Ads Tools (`google_ads_tools/`)

Utility scripts for Google Ads data processing:

- `generate_google_ads_imports.py` - Generate Google Ads import files
- `fix_google_ads_templates.py` - Fix and validate Google Ads templates

## Usage

These scripts are intended for specific analysis scenarios and may require configuration before use. Ensure proper environment variables and credentials are set up before running.
