# Geo Performance Analyzer Claude Skill

## Overview
Analyzes location-based performance for Google Ads campaigns and recommends geographic bid adjustments optimized for retail businesses with physical stores.

## Business Value
- **Typical Impact**: 15-25% ROAS improvement
- **Cost Savings**: 10-15% from bid adjustments and exclusions
- **Revenue Increase**: 10-20% from bid ups on top locations
- **Best For**: Retail businesses, service area businesses, multi-location brands

## What It Does
1. Identifies top and bottom performing locations
2. Recommends bid adjustments (+/-20% to +/-50%)
3. Suggests location exclusions for waste elimination
4. Correlates performance with store proximity (if applicable)
5. Analyzes local intent search patterns

## Required MCP Tools
- `get_geo_performance` - Location-level performance data

## Key Features
- Account average comparison for relative performance
- Retail-specific store proximity analysis
- "Near me" search performance tracking
- Drive-time considerations for local businesses

## How to Use
```
Analyze geographic performance for customer ID 1234567890
over the last 90 days and recommend location bid adjustments.
```
